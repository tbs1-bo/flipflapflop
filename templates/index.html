Das Display kann aus der Ferne gesteuert werden.

Dimension: {{ dimension }}

{{ request.host }}/px/x/y/onoff
setzt (onoff ist 'on') oder löscht (onoff ist 'off') einen Pixel an der Stelle (x|y): http://{{ request.host }}/px/3/2/on

{{ request.host }}/page
erwartet einen POST-Request, welcher aus 0en, 1en oder xen besteht, die wiederum die Pixel setzen (1) oder löschen (0) oder den Wert nicht ändern (x): $ curl -d 110110 {{ request.host }}/page

{{ request.host }}/txt
erwartet einen POST-Request, welcher einen Text enthält, der auf dem Display dargestellt werden soll: $ curl -d Hallo {{ request.host }}/txt
