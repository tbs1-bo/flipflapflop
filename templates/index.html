{{ request.host }}/px/x/y/onoff setzt (onoff ist 'on') oder löscht (onoff ist 'off') einen Pixel an der Stelle (x|y): http://{{ request.host }}/px/3/2/on

{{ request.host }}/page erwartet einen POST-Request mit dem Parameter data, welcher aus 0en und 1en besteht, die wiederum die Pixel setzen (1) oder löschen (0): $ curl -d data=110110 {{ request.host }}/page

{{ request.host }}/txt erwartet einen POST-Request mit dem Parameter data, welcher einen Text enthält, der auf dem Display dargestellt werden soll: $ curl -d data=Hallo {{ request.host }}/txt
